# Home Assistant Blueprints

Public collection of Home Assistant automation blueprints.

## Blueprints

See individual blueprint directories for documentation and usage.

## Installation

1. Navigate to your Home Assistant configuration directory
2. Add this repository as a submodule:
   ```bash
   git submodule add https://github.com/isaackehle/homeassistant-blueprints.git blueprints
   ```

## Usage

Import blueprints in Home Assistant UI:
- Settings → Automations & Scenes → Blueprints → Import Blueprint
- Use the file path from this repository

## Contributing
